# POS Kiosk-Based Inventory System

## Project Overview
This is a kiosk-based point-of-sale (POS) and inventory system tailored for managing school supplies. It allows customers to self-order via a touchscreen kiosk and provides an admin panel for managing inventory, tracking sales, logging system activity, and generating reports.
